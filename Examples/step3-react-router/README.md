## 핵심 내용

- React 멀티 페이지 구현을 위한 React Router에 대한 이해
- React Router의 여러 훅 사용(Routes, Route, Link, useNavigate, useParams, useSearchParams, Navigate)에 대한 이해
- Restful Routes에 대한 이해
- Redirect에 대한 이해

## 시작 가이드

- 해당 프로젝트 폴더에서 터미널 열기

```bash
npm install
```

- install 후 실행

```bash
npm start
```

## 사용 npm

- react-router

```bash
npm install react-router@6 react-router-dom@6
```
